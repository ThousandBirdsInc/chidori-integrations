# @chidori/integrations-openai-compatible

OpenAI-compatible provider tools for Chidori agents.

Exports named tools for:

- xAI
- Fireworks AI chat, completions, and embeddings
- Together AI
- Perplexity Agent API
- DeepSeek
- Hugging Face Inference Providers chat and embeddings
- DeepInfra
- ZhipuAI
- Moonshot AI
- Alibaba Cloud Model Studio DashScope
- Tencent Hunyuan chat and embeddings
- ByteDance Volcengine Ark / Doubao
- MiniMax chat and embeddings
- Novita AI
- Friendli
- SambaNova
- PremAI chat and embeddings
- Upstage Solar chat and embeddings
- GitHub Models
- Cerebras
- NVIDIA NIM

The tools call HTTP endpoints directly through `chidori.http` and do not require provider SDKs.

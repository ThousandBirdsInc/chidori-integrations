# @chidori/integrations-composio

Chidori tools for Composio's REST tool platform API.

Exports:

- `composioToolsListTool`
- `composioToolGetTool`
- `composioToolExecuteTool`
- `composioProxyExecuteTool`
- `composioTools`

These wrappers use Composio's HTTP API directly and do not require Composio SDK packages inside a Chidori agent.

# @chidori/integrations-replicate

Replicate prediction tools for Chidori agents.

Exports:

- `replicatePredictionCreateTool`
- `replicateOfficialModelPredictionCreateTool`
- `replicateDeploymentPredictionCreateTool`
- `replicatePredictionGetTool`
- `replicatePredictionCancelTool`
- `replicateTools`

The tools call Replicate's HTTP API directly through `chidori.http` and do not require the Replicate SDK.
